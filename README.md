# Dahhan Landing (Next.js)

## Setup
1. Copy `.env.example` to `.env` and set DB/SMTP values.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev:
   ```bash
   npm run dev
   ```

## Pages
- `/ar` Arabic landing
- `/en` English landing
- `/admin` Admin dashboard

## Notes
- Images are served from `public/images` and uploads go to `public/uploads`.
- The admin password is stored in DB (`cfgs_wpass`).
